
# 🏛️ Roman Number Converter

A lightweight Python script that transforms Roman numerals into their decimal form. Perfect for quick lookups or playing ancient number wizard from your terminal.

---

## 💡 Features

- Converts Roman numerals with support for common subtractive notation (e.g., IV, XL, CM)
- Interactive CLI prompt with a themed welcome message
- Requires only Python

---

## ▶️ Usage

Run the script in your terminal:

```bash
python roman_converter.py
```

Enter a Roman numeral (like `MCMXCIV`), and get your result instantly.
